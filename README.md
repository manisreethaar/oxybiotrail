# Oxygen BioInnovations

**Nature's Blueprint. Perfected by Science.**

Oxygen BioInnovations is a biotechnology company focused on sustainable solutions, functional foods, and biocosmetics. We bridge the gap between biological research and consumer needs.

## Project Overview

This website is built with a modern tech stack to reflect our "High-Tech Organic" identity.

### Tech Stack
-   **Framework**: React (Vite)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS
-   **UI Components**: Shadcn UI
-   **Routing**: React Router DOM

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/manisreethaar/oxybiotrail.git
    cd oxybiotrail
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
